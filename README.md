# Concepts Practiced

This project focuses on essential React fundamentals, including:

- React components & JSX
- Props and component reusability
- Handling events and user input
- Two-way data binding
- Lifting state up
- State management with hooks
- Derived & computed values
- Conditional rendering
- Rendering dynamic lists
- Formatting numeric values

# Features

- User input for:
  - Initial investment
  - Annual investment
  - Expected return
  - Investment duration
- Real-time calculation of investment growth
- Dynamic results table
- Input validation (prevents zero or negative values)
- Clean and readable UI

---

# Built With

- React
- Vite
- JavaScript (ES6+)
- CSS


# How It Works

1. The user enters investment details.
2. Input values are managed via lifted state.
3. Investment results are calculated dynamically.
4. Results are displayed in a table for each year.
5. Invalid inputs trigger an error message.
